UT4_V2_TA6_EJ2

Dataset: TA6-EJ2.csv
In this exercise, we used a small dataset with 17 examples, featuring two predictors (X1, X2) and a binary output labeled as Class A or Class B.

Process Overview:
The process implemented in RapidMiner involved importing the dataset, assigning roles using the "Set Role" operator, and training an SVM model. We used the "Apply Model" operator to evaluate test data.

Preparation:
1. Imported the dataset "TA6-EJ2.csv".
2. Used "Set Role" to define the "class" attribute as the prediction target.
3. Created the SVM model and added test points: (1.5, 1), (1.5, 4), and (2, 7).

Kernel Model:
The kernel model parameters provided details about the weights and offset of the SVM. These weights were:
- w[X1] = 0.370
- w[X2] = 1.351
The model used a total of 17 support vectors to define the decision boundary.

Results:
Predictions were made for the test points based on the trained SVM model. The confidence scores indicated the model's certainty for each prediction.

Prediction Table:
The model provided the following predictions:
- Point (1.5, 1): Class A, Confidence 87.9%
- Point (1.5, 4): Class B, Confidence 53.4%
- Point (2, 7): Class B, Confidence 92.6%
These results demonstrate how the SVM uses the decision boundary to classify test points.

Conclusion:
The SVM implementation successfully classified new test points with high confidence. The kernel model and support vectors ensured accurate predictions, validating the model's effectiveness for this simple binary classification problem.
